  THE MET Museum Landing Page
Welcome to the landing page for the Met Museum, a showcase of its available collections, visiting hours, and contacts. Built using the BEM methodology and the SCSS preprocessor, this page ensures clean and maintainable code. The layout is adaptive and responsive, providing a seamless user experience across devices of different screen sizes.

  Technologies Used:
HTML
CSS (SCSS preprocessor)
JavaScript

  Folder Structure
- index.html: Main HTML file
- /styles: Contains the SCSS source files
- /scripts: Contains the JavaScript files
- /images: Contains the images used on the page

  Features
1. Clean and Maintainable Code: Utilizes the BEM methodology and SCSS preprocessor.
2. Adaptive and Responsive Layout: Ensures a seamless experience across different devices.
3. High-Quality Images: Showcases the museum's art collections.
4. Responsive Navigation Menu: Adapts to different screen sizes.
